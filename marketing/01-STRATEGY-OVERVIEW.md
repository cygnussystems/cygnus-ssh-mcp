# cygnus-ssh-mcp Marketing Strategy

## The Core Problem

You have a **legitimately excellent product** with **zero discovery**. The README is professional, the feature set is comprehensive (43 tools!), and the cross-platform support is a genuine differentiator. But nobody knows it exists.

## Your Unique Selling Points

1. **True cross-platform SSH** - Windows ↔ Linux ↔ macOS in any direction
2. **43 specialized tools** vs competitors' "just run commands"
3. **Line-level file editing** - surgical precision, no download/upload
4. **Background task management** - launch, monitor, kill
5. **Full Unicode support** - actually works, even on Windows
6. **Pre-configured hosts with aliases** - "connect to prod" not credentials every time

## Target Audiences

| Audience | Pain Point | Your Solution |
|----------|------------|---------------|
| **DevOps engineers** | Managing multiple servers via Claude is clunky | Host aliases, sudo support, background tasks |
| **Developers with remote dev boxes** | Can't use Claude Code on beefy Linux servers | Full remote control from any OS |
| **Sysadmins** | Repetitive server tasks | AI-assisted automation |
| **Homelabbers** | Managing Raspberry Pis, NAS boxes | Easy multi-server control |

## Primary Message

> **"Give Claude full control of any server - Linux, Windows, macOS"**

or

> **"The only SSH MCP that actually works cross-platform"**

## Marketing Funnel

```
Discovery (HN, Reddit, MCP Registry)
    ↓
README (5-second value prop)
    ↓
Quick Start (2-minute setup)
    ↓
First Success ("connect to prod and show disk usage")
    ↓
Star + Share
```

## Success Metrics

- **Week 1 goal**: 50 stars, 10 real users
- **Month 1 goal**: 200 stars, listed in MCP Registry
- **Month 3 goal**: 500+ stars, community contributions

## Budget: $0

All tactics in this playbook are free. You're trading time, not money.
