# cygnus-ssh-mcp

**The most powerful SSH MCP server for AI assistants** — give Claude full control of
your Linux, macOS, and Windows servers with 46 specialized tools.

This site covers detailed usage docs. For installation quick-start, feature
highlights, and the full tool list, see the
[project README](https://github.com/cygnussystems/cygnus-ssh-mcp) on GitHub.
